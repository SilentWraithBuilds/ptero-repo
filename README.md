# 🦖 PTERODACTYL INSTALL SCRIPT - SILENT WRAITH

A one-command installer for Pterodactyl Panel, Wings, and Tailscale.

## 🚀 Features

* Install Panel (auto configured)
* Install Wings (interactive config)
* Install Tailscale
* Auto queue worker
* Auto systemd services
* Firewall setup
* Clean UI with progress bar

## ⚡ Installation

Run this command:

```bash
bash <(curl -s https://raw.githubusercontent.com/SilentWraithBuilds/ptero-repo/main/install.sh)
```

## ⚠️ Requirements

* Fresh VPS (Debian 12 / Ubuntu 22.04+)
* Root access
* Domain ready (for panel)

## 👤 Default Panel Login

* Username: admin
* Password: admin

## 📌 Notes

* Designed for Cloudflare Tunnel compatibility
* Not recommended on already-used systems
